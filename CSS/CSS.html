定位

position是定位 其值fixed為固定定位

可以將其固定在畫面上 如同右下的方框(須加上右修飾與下修飾)

這將會使其失去原本的位置(如float)


relative為相對定位 會相對於原本該顯示的地方偏移

此方法只是改變顯示定位 原本所佔的空間依然存在


absolute為絕對定位 與fixed相同 會失去原本的位置

絕對定位是根據網頁來決定位置 而非fixed的畫面位置(如本網頁最左上角)
